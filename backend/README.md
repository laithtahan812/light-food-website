# ЛАЙТ FOOD — Backend (Express + SQLite)

## Быстрый старт
1) Установите Node.js 18+  
2) Скопируйте `.env.example` в `.env` и задайте `JWT_SECRET`  
3) Установите зависимости и создайте БД с тестовым меню:

```bash
cd backend
npm install
npm run seed
npm run dev
```

Сервер будет доступен по адресу: `http://localhost:4000`
